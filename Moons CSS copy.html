Hello CSS! Part 1

CSS stands for Cascading Style Sheets.


There are three types in CSS
  1. Inline CSS

    2. Inline CSS uses the style attribute in HTML element

  2. Internal CSS

    3. Internal CSS uses a style tag element in the head section

  3. External CSS

    4. External CSS uses an external CSS file

External CSS

External CSS is used to define the style for many HTML pages


To use an External style sheet, add a link to it in the head section of the HTML page


An external style sheet can be written in any text editor.
The file must not contain any HTML code, and must be saved with a .css extension.
